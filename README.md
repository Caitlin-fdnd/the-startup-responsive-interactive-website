## Interacties

Ik heb 2 interacties gecodeerd, een selectie menu waarin meerde acties gedaan kunnen worden en een paar cards waar op gehoverd en geklikt kan worden, beide interacties hebben een animatie die in een "prefers reduced motion" media query zitten wat betekent dat als de gebruiker animaties op het apparaat uit heeft geschakeld de animaties niet activeren.

## Responsive

Mijn hele website is responsive gemaakt aan de hand van een mobile first layout en daarna media queries to te voegen zodat de website ook op desktop bekeken kan worden.

## Toegankelijkheid

Ik heb mijn website getest aan de hand van de WCAG test om ervoor te zorgen dat hij volledig bruikbaar is voor gebruiker met screenreader, gebruikers die visueel beperkt zijn en gebruikers die het toetsenbord gebruiken om te navigeren.
